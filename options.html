Como a integração funciona?
A extensão envia o endereço bruto para o seu backend em Go. O backend formata o endereço com Claude e usa Nominatim quando precisar preencher latitude, longitude e bairro.

⚠️ Aviso Importante
A URL do backend deve estar acessível pelo navegador e, se você usar autenticação, o token é armazenado localmente no seu navegador.

Exemplo local: http://localhost:8080. Ajuste para o host onde seu serviço Go estiver rodando.
Opcional. Se o backend exigir autenticação, este valor será enviado como Bearer token.
